\section{Progress report}

We have not yet been able to verify the claims made in the article. We
have built a basic framework in Python to apply the Data degradation
to the dataset, but it does not yet do any useful calculations on the
risk and worth values for the data records.\\

Next, we plan to do the following:


\begin{enumerate}

\item Implement the {\tt totworth}, {\tt risk}, and {\tt priv}
  functions defined in the article

\item Apply the functions to each record in the dataset with different
  arguments

\item Choose the best arguments based on the article and the experiments

\item Apply the degradation with different levels

\item Compare the output of different levels of degradation and count
  the risk and worth for all of them

\item Compare the results obtained to the ones mentioned in the
  article.

\end{enumerate}
